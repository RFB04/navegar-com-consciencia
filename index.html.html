Segurança, Inclusão e Cidadania Digital

1. O que é Educação Digital?

Uso de tecnologias para ensinar, aprender e compartilhar conhecimento. Inclui EAD, apps, videoaulas, etc.

Atenção: Deve ser segura e inclusiva.

2. Dicas de Segurança para Navegar na Internet

  • Crie senhas seguras com letras, números e símbolos.
  • Não compartilhe dados pessoais com desconhecidos.
  • Mantenha seu antivírus atualizado.
  • Evite clicar em links suspeitos.
  • Verifique se o site possui conexão segura.

3. O que é LGPD?

Lei Geral de Proteção de Dados garante que seus dados sejam tratados com responsabilidade.

  • Você pode saber quais dados estão sendo coletados.
  • Permitir ou negar o uso de dados.
  • Solicitar exclusão de dados.
